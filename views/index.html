Proyecto Api, Acortador de URLs

Introducimos la url que deseamos acortar, la enviamos al servidor y este responde con un json donde se encuentra la url acortada, el servidor la relaciona a la url original en la bd

Creacion de la URL

para utilizarla

[Este Url]/api/shorturl/[ingresa aqui el url acortado]

y los redireccionara a el url original
